To provide configuration for the base image styles used throughout Sitefarm features.

* Gallery Full
* Gallery Thumbnail
* Hero Banner
* Landscape 16x9
* Landscape 4x3
* Large Width (1000px)
* Medium Width (500px)
* Profile
* Slideshow Full
* Slideshow Thumbnail
* Small width (200px)
* Thumbnail
* Title Banner
